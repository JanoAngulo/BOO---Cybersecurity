
  # BOO - Cybersecurity

  BOO - Cybersecurity is a project focused on building secure, scalable, and user-friendly solutions to protect applications and data from modern cyber threats. It combines strong security practices with clean UX/UI design principles to ensure that security is not only effective but also accessible to end-users.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
